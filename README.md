## Что добавлено в форке?

### Экспорт анимации по кадрам в файл

Добавлена возможность сохранять полученные анимации в текстовый файл по кадрам.

Пример использования:  
```bash
python cli.py -f example.gif -n output.txt
```   
Важные моменты:
- Параметр -n или --name_of_output определяет название файла для записи. 
- По умолчанию используется расширение .txt
- Запись не производится, если параметр не указан.

Кадры разделены друг от друга строкой в 50 тире для удобства парсинга.

Установка:
``` bash
git clone https://github.com/TamerLAN42/ascii-animator.git
cd ascii-animator
pip install -r requirements.txt
```
